# Requisitos Não Funcionais
#### A _Tabela 2_ a seguir contém os **Requisitos Não Funcionais (RNF)** elicitados.

| ID   |             Requisitos              | Categoria e Tipo |
| :--: | :---------------------------------: | :--------------: |
| RNF01 | A interface do sistema deve ser extremamente intuitiva, permitindo que um usuário com pouca experiência tecnológica aprenda a realizar as operações principais em menos de 10 minutos de uso.  | Usabilidade |
| RNF02 | Todos os textos da interface devem ter um tamanho de fonte legível e os elementos interativos (botões, ícones) devem ser grandes, identificáveis e com alto contraste de cores. | Usabilidade |
| RNF03 | O sistema de alertas deve ter uma taxa de sucesso de 100% para notificações programadas enquanto o dispositivo estiver operacional e a aplicação em execução. O sistema não pode pular um alerta. | Confiabilidade |
| RNF04 | O sistema deve garantir a integridade dos dados do usuário. Em caso de fechamento inesperado da aplicação, os dados já salvos não devem ser corrompidos. | Confiabilidade |
| RNF05 | A aplicação deve ter um tempo de inicialização rápido em dispositivos compatíveis com as configurações mínimas recomendadas. | Desempenho |
| RNF06 | A resposta a qualquer ação do usuário na interface (cliques, toques, etc.) deve ser praticamente instantânea, não excedendo 1 segundo.  | Desempenho |
| RNF07 | Os dados do usuário devem ser armazenados de forma segura e protegidos por um mecanismo de login (usuário e senha/PIN), impedindo o acesso não autorizado no dispositivo.  | Segurança |
| RNF08 | O sistema deve ser compatível com as versões de sistemas operacionais modernos e mais prevalentes entre o público-alvo. | Compatibilidade |
| RNF09 | O processo de instalação/obtenção da aplicação deve ser simples e direto, guiando o usuário com o mínimo de passos possível. | Instalação |
| RNF10 | O código-fonte deve seguir boas práticas de desenvolvimento e ser claramente documentado para facilitar futuras manutenções e evoluções do produto. | Manutenibilidade |
